# Email Integration Docs

Use these docs in this order:

- [Email SDK Integration](email-README.md) - reusable KDCube SDK modules,
  bundle wiring, account store, provider access, delivery helpers, attachment
  materialization, Email MCP, and Claude Code processing.
- [Email External Prerequisites](email-external-prereq-README.md) - work that
  must happen outside KDCube before the integration can function, including
  Google Cloud/Gmail API setup, OAuth callback URLs, deployment secrets, and
  iCloud app-password requirements.
