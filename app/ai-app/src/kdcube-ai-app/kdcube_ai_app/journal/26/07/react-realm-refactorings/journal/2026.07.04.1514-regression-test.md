## Clean index
• Use a transaction so you can inspect the count before committing.
```sql
BEGIN;

SELECT count(*) AS rows_to_delete
FROM kdcube_demo_tenant_demo_project.conv_messages
WHERE user_id = '02e53484-0081-70ce-11c1-e96706b1a182';

DELETE FROM kdcube_demo_tenant_demo_project.conv_messages
WHERE user_id = '02e53484-0081-70ce-11c1-e96706b1a182';

-- Verify it is clean:
SELECT count(*) AS remaining_rows
FROM kdcube_demo_tenant_demo_project.conv_messages
WHERE user_id = '02e53484-0081-70ce-11c1-e96706b1a182';

-- If correct:
COMMIT;

-- If not correct, run instead of COMMIT:
-- ROLLBACK;
```

If your local schema differs, replace kdcube_demo_tenant_demo_project with the actual tenant/project schema.
